Proyecto ambicioso que combina narrativa profunda, sistemas interactivos y arte visual único. Implementé un sistema de combate por turnos, IA estratégica para jefe final, sistema de decisiones con múltiples finales y mezcla 2D/3D en estilo 2.5D.

Mi mayor logro técnico fue el diseño e implementación de una lógica de decisión avanzada para la IA del jefe. Desarrollé un sistema que evalúa dinámicamente múltiples variables (estado, vida, defensa, prioridades y condiciones simultáneas) y toma decisiones estratégicas en tiempo real utilizando estructuras enlazadas y comparaciones condicionales complejas. La IA no actúa de forma aleatoria simple, sino que analiza el contexto completo antes de ejecutar una acción, incorporando además probabilidades controladas para generar comportamientos variables pero coherentes.


También diseñé un sistema completo de gestión de turnos basado en estados, controlando transiciones, validaciones, bloqueos de entrada, ejecución asincrónica mediante coroutines y sincronización con animaciones y eventos. Implementé control de flujo robusto para evitar estados inválidos, gestión de recursos (vida, energía, objetos), verificación de condiciones límite y control automático del ciclo principal del sistema.


Este proyecto demuestra que puedo desarrollar sistemas con lógica compleja, manejar flujos de ejecución, y crear soluciones modulares que se adaptan a distintas situaciones.

Unvollendet – Sistema avanzado de juego por turnos

Desarrollé Unvollendet, un proyecto de videojuego ambicioso que combina narrativa profunda, sistemas interactivos y arte visual único. Implementé un sistema de combate por turnos donde cada personaje tiene habilidades, tiempos de acción y animaciones propias, y desarrollé una IA avanzada para el jefe que toma decisiones estratégicas considerando la salud y defensa de los jugadores, así como elementos aleatorios para crear retos dinámicos.

El juego integra escenas con actores reales y narrativa tipo novela gráfica, combinando elementos 2D y 3D en un estilo 2.5D, y permite a los jugadores tomar decisiones que desbloquean distintos finales. Este proyecto demuestra mi capacidad de diseñar sistemas complejos, estructurar código modular, implementar lógica de IA y gestionar interacciones entre múltiples componentes.
