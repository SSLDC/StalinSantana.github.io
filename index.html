Unvollendet – Sistema avanzado de juego por turnos

Proyecto ambicioso que combina narrativa, sistemas jugables e integración 2D/3D en un entorno 2.5D. Implementé un sistema de combate por turnos, IA estratégica para un jefe final y un sistema de decisiones con múltiples finales.

Mi mayor trabajo técnico fue la IA del jefe. Diseñé un sistema que analiza en tiempo real variables como vida, defensa, estado del combate y prioridades, y toma decisiones en base a ese contexto. No es una IA aleatoria: evalúa la situación antes de actuar e incluye probabilidades controladas para generar comportamientos variables pero coherentes.

También desarrollé un sistema de turnos basado en estados, con control de transiciones, validaciones, bloqueo de entradas y ejecución asincrónica mediante coroutines. Todo el flujo del combate está controlado para evitar estados inválidos y asegurar coherencia entre lógica, animaciones y eventos.

Este proyecto demuestra mi capacidad para crear sistemas complejos, estructurar lógica de juego y construir soluciones modulares que se adaptan a diferentes situaciones.

Sistema de movimientos, físicas y desarrollo de herramientas personalizadas

Desarrollé una demo técnica centrada en arquitectura modular, control de estados complejos y sistemas desacoplados. Separé claramente la lógica de entrada, físicas, cámara y comportamiento para mantener un código escalable y organizado.

El sistema incluye estados avanzados como dash, salto variable, caída, colisiones laterales y corrección de esquinas. Implementé control de flujo basado en condiciones y eventos, con interpolaciones temporizadas mediante coroutines y ajustes dinámicos de parámetros como la cámara según la velocidad del personaje.

Además, creé herramientas personalizadas en el editor de Unity que permiten configurar cámaras y parámetros directamente desde el inspector sin modificar código. Esto demuestra capacidad para diseñar sistemas reutilizables y herramientas internas que mejoran el flujo de trabajo.

Este proyecto refleja mi enfoque en código limpio, escalable y preparado para casos límite reales.
